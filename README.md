# Проект java-explore-with-me

Проект java-explore-with-me — это многомодульное приложение на Spring Boot, реализующее функционал платформы для поиска и организации мероприятий (аналог "Explore With Me").

## Основные модули проекта

* **main-service** — основной сервис, реализующий логику приложения: события, категории, пользователи, запросы на участие и т. д.
* **stats-service** — сервис для сбора и обработки статистики посещений.


## Основные функции

* Создание и управление событиями — пользователи могут создавать мероприятия с описанием, датой, локацией и т. д.
* Категории мероприятий — события можно классифицировать по категориям
* Комментарии к событиям — добавлена новая функциональность для комментирования событий
* Статистика посещений — сбор и анализ данных о посещении мероприятий
* Управление пользователями — регистрация, авторизация и профили пользователей

## Технологии

* Spring Boot 3.3.2
* Java 21
* Maven
* JUnit 5, Mockito
* Lombok
* Checkstyle, SpotBugs, JaCoCo

## Архитектура

Проект реализует паттерн микросервисной архитектуры с использованием современных практик разработки и инструментов.
